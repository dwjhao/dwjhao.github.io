#目录结构

##安装Node.js
nodejs.org
node-v\nmp-v 查看版本

##安装Git
git-scm.com (最后一步添加路径时选择Use Git from the Windows Command Prompt)
git –version

##注册Github账号
github.com
新建项目： 用户名.github.io

##安装Hexo
新建文件右键打开Git Bash Here控制窗口
安装 npm i hexo-cli -g
验证 hexo -v
初始化 hexo init
安装组件 npm install
hexo g 生成静态网页
hexo s 打开本地服务器，然后浏览器打开http://localhost:4000/
hexo clean 清除缓存
hexo d 上传到github

###连接Github与本地
git config –global user.name “dwjhao”
git config –global user.email “1247520890@qq.com“
查看 git config –global user.name
git config –global user.email

生成ssh key ：ssh-keygen -t rsa -C "1247520890@qq.com"
github settings中 新建一个SSH 名字随便 输入.ssh文件里的id_rsa.pub（文本记事器参看）
查看ssh -T git@github.com  出现用户名成功     
  更改_Cconfig.yml
     最后一行：（小心谨慎）
      deploy:
        type: git
        repository: https://github.com/dwjhao/dwjhao.github.io
        branch: master

###写文章、发布文章
在source/_posts 新建.md文件
hexo g 生成静态文档
hexo d 上传到github

###主题修改
在 hexo.io/themes/index.html 找到喜欢的主题 里面有安装方法： git clone https://github.com/huweihuang/hexo-theme-huweihuang.git ./hexo-huweihuang
在 _Cconfig.yml 更改themes为themes文件库所选主题名

google 快捷键

以下是我总结的有用的几个

###标签页和窗口快捷键：

  1. 新窗口 打开：ctrl+n 无痕：ctrl+shift+t

    关闭：atrl+shift+w | alt +F4
 切换：alt+tab

  2. 新标签页 打开：ctrl+t 之前关闭的标签页：ctrl+shift+t

    关闭：ctrl+w | ctrl+F4
 切换： 上下： ctrl+tab | ctrl+shift+tab
        指定： ctrl+1——ctrl+8
         主页：ctrl+home

###功能

  1. 菜单： atl+f | alt+e
  2. 书签： ctrl+shift+b 管理器： ctrl+shift+o 保存：Ctrl + d
  3. 任务管理器： shift+esc
  4. 打开搜索栏： ctrl+f
  5. 开发者工具： F12 | ctrl+shift+j
  6. 跳转到与查找栏中搜索字词相匹配的下|上一条内容： Ctrl + g | Ctrl + Shift + g
  7. 清除缓存： ctrl+shift+delete

atom的使用心得

Share

快捷键

  1. 命令面板： Ctrl+Shift+P 模糊搜索Atom中和插件中定义的所有命令
  2. 打开文件：Ctrl+O 文件夹：Ctrl+shift+O
  3. 目录树栏：Ctrl+\
  4. 搜索目录中的文件： Ctrl+T | Ctrl+P
  5. 搜索一个当前打开的文件：Ctrl+B
  6. 搜索一个新建的或更改过的文件：Ctrl+Shift+B

插件

安装（ settings | cmd）

文本编辑

*整行文本交换： atrl+up|down
*光标快速移动： 行：列
*整行选择： shift+up | down
*复制当前行：Cmd+Shift+D
*删除当前行: Ctrl+Shift+K
*多光标：ctrl+鼠标点击位置

代码片段

tab

snippet:

  1. ‘source.js’
  2. ‘console.log’
  3. ‘predix’:’log’
  4. ‘body’:’console.log(${1:”crash”});$2’
    使用： console.log(“crash”)
  • 自动补全插件：autocomplete-plus
  • 代码折叠
    Ctrl + Alt + Shift + { : 折叠（fold）所有代码
    Ctrl + Alt + Shift + } : 打开（unfold）所有代码
  • 预览： ctrl+shift+m
  • 代码类型： ctrl+shift+L
  • 拼写检查： Ctrl+Shift+;
  • 开发工具： Ctrl+Shift+I
  • Markdown-Writer 插件编写.md文件
