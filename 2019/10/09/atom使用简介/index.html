atom的使用心得

Share

快捷键

  1. 命令面板： Ctrl+Shift+P 模糊搜索Atom中和插件中定义的所有命令
  2. 打开文件：Ctrl+O 文件夹：Ctrl+shift+O
  3. 目录树栏：Ctrl+\
  4. 搜索目录中的文件： Ctrl+T | Ctrl+P
  5. 搜索一个当前打开的文件：Ctrl+B
  6. 搜索一个新建的或更改过的文件：Ctrl+Shift+B

插件

安装（ settings | cmd）

文本编辑

*整行文本交换： atrl+up|down
*光标快速移动： 行：列
*整行选择： shift+up | down
*复制当前行：Cmd+Shift+D
*删除当前行: Ctrl+Shift+K
*多光标：ctrl+鼠标点击位置

代码片段

tab

snippet:

  1. ‘source.js’
  2. ‘console.log’
  3. ‘predix’:’log’
  4. ‘body’:’console.log(${1:”crash”});$2’
    使用： console.log(“crash”)
  • 自动补全插件：autocomplete-plus
  • 代码折叠
    Ctrl + Alt + Shift + { : 折叠（fold）所有代码
    Ctrl + Alt + Shift + } : 打开（unfold）所有代码
  • 预览： ctrl+shift+m
  • 代码类型： ctrl+shift+L
  • 拼写检查： Ctrl+Shift+;
  • 开发工具： Ctrl+Shift+I
  • Markdown-Writer 插件编写.md文件
