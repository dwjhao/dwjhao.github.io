#目录结构

##安装Node.js
nodejs.org
node-v\nmp-v 查看版本

##安装Git
git-scm.com (最后一步添加路径时选择Use Git from the Windows Command Prompt)
git –version

##注册Github账号
github.com
新建项目： 用户名.github.io

##安装Hexo
新建文件右键打开Git Bash Here控制窗口
安装 npm i hexo-cli -g
验证 hexo -v
初始化 hexo init
安装组件 npm install
hexo g 生成静态网页
hexo s 打开本地服务器，然后浏览器打开http://localhost:4000/
hexo clean 清除缓存
hexo d 上传到github

###连接Github与本地
git config –global user.name “dwjhao”
git config –global user.email “1247520890@qq.com“
查看 git config –global user.name
git config –global user.email

生成ssh key ：ssh-keygen -t rsa -C "1247520890@qq.com"
github settings中 新建一个SSH 名字随便 输入.ssh文件里的id_rsa.pub（文本记事器参看）
查看ssh -T git@github.com  出现用户名成功     
  更改_Cconfig.yml
     最后一行：（小心谨慎）
      deploy:
        type: git
        repository: https://github.com/dwjhao/dwjhao.github.io
        branch: master

###写文章、发布文章
在source/_posts 新建.md文件
hexo g 生成静态文档
hexo d 上传到github

###主题修改
在 hexo.io/themes/index.html 找到喜欢的主题 里面有安装方法： git clone https://github.com/huweihuang/hexo-theme-huweihuang.git ./hexo-huweihuang
在 _Cconfig.yml 更改themes为themes文件库所选主题名
